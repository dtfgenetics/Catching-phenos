# Build Roadmap

## Phase 0 — Pre-Production Structure

Goal: create the repository foundation, design docs, schemas, and MVP data files.

Deliverables:

- Game Bible
- MVP Master Sheet
- Codex Build Brief
- system design documents
- world design documents
- data schemas
- starter data files
- asset manifest
- initial web scaffold

## Phase 1 — Data-Driven Browser Prototype

Goal: build the smallest playable loop using placeholder art.

Deliverables:

- web entry point
- game config
- data loader
- local save/load
- dialogue engine
- basic map screen
- encounter trigger
- battle screen
- recipe timer screen
- PhenoLog screen

## Phase 2 — MVP Game Loop

Goal: complete the Seedling Town Demo loop.

Deliverables:

- starter selection
- Terp Fields encounter table
- battle reward flow
- timer-based unit creation
- first Vault progress update
- Team Lockout first event
- Aroma Trial event
- demo ending

## Phase 3 — Art Replacement

Goal: replace placeholders with original game assets.

Deliverables:

- player sprite
- NPC sprites
- first Pheno sprites
- UI panels
- tilesets
- title screen
- simple audio

## Phase 4 — Advanced Systems

Goal: expand the prototype into the full game structure.

Deliverables:

- full expression matrix
- genotype scanner
- Lineage Lab
- batch result system
- region restoration
- Vault Garden expansion
- daily and weekly website events

## Phase 5 — DTF Seeds Integration

Goal: host the game as a DTF Seeds browser game.

Deliverables:

- route-ready public build
- responsive layout
- mobile controls
- landing page copy
- disclaimer
- update log
- community hooks
