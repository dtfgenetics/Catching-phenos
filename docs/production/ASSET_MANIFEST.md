# Asset Manifest

This file tracks the assets needed for the first playable PhenoQuest demo.

## Asset Rules

- All art must be original.
- Do not copy protected monster-game logos, character silhouettes, capture-device imagery, fonts, UI layout, music, or sound effects.
- Use transparent PNG for sprites and UI layers where possible.
- Keep source files separate from web-export files when added later.

## Recommended Sizes

| Asset Type | Recommended Size |
|---|---:|
| Player overworld frame | 32x32 or 48x48 |
| NPC overworld frame | 32x32 or 48x48 |
| Pheno overworld icon | 32x32 |
| Pheno battle sprite | 128x128 |
| Boss battle sprite | 192x192 |
| Item icon | 32x32 |
| UI icon | 32x32 |
| PhenoLog portrait | 96x96 |
| Tiles | 16x16 or 32x32 |
