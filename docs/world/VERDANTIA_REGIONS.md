# Verdantia Regions

## World Purpose

Verdantia is the main fantasy region for PhenoQuest. Each region teaches a system, unlocks a Garden Seal, contains unique expression conditions, and restores a branch of the Living Seed Vault.

## Region Progression

1. Seedling Town
2. Terp Fields
3. Kushwood Forest
4. Gas Cave
5. Frost Peak
6. Clone District
7. Living Seed Vault

## Seedling Town

Tutorial hub and home base.

### Key Locations

- Player House
- Professor Calyx Greenhouse
- Cure Station
- Seed Supply Shop
- Vault Gate
- Training Patch
- Terp Fields exit

### Systems Introduced

- movement
- dialogue
- starter selection
- first battle
- PhenoLog
- Clone Dome intro

## Kushwood Forest

Second major region.

### Theme

Roots, structure, stability, old knowledge, common genetics mattering.

### Systems Introduced

- stronger Stability management
- Root-class interactions
- Keeper comparison
- early genotype hints

## Gas Cave

Power and risk region.

### Theme

Heavy force, resin energy, unstable strength, resource extraction.

### Systems Introduced

- high Power / low Stability tradeoffs
- Lockout machines
- risk-based expression outcomes

## Frost Peak

Rare expression and Vault mystery region.

### Theme

Resin, cold expression, Legacy clues, patience.

### Systems Introduced

- rare expression hunting
- Legacy Echo hints
- advanced PhenoLog records

## Clone District

Team Lockout-controlled city.

### Theme

Sterile perfection, synthetic control, artificial sameness.

### Systems Introduced

- corrupted material
- synthetic enemies
- cleansing systems
- Nova story turning point

## Living Seed Vault

Final hub and endgame region.

### Theme

Restoration, lineage memory, final confrontation, Legacy Seed.

### Systems Introduced

- full Vault branch restoration
- Null Cultivar confrontation
- Legacy progression
- lineage completion rewards

## Region Restoration System

Each region should track restoration percentage.

Possible unlock thresholds:

| Progress | Unlock |
|---|---|
| 10% | NPC dialogue updates |
| 25% | new spawn appears |
| 50% | environmental effect unlocks |
| 75% | rare expression chance increases |
| 100% | Vault Branch restored |
