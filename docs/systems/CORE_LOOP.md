# Core Gameplay Loop System

## Core Loop Statement

**Battle for Cuttings. Root your Clones. Find your Keepers. Read the Genotype. Make the Cross. Grow the Expression. Restore the Lineage.**

This is the backbone of PhenoQuest.

## Primary Loop

1. Player explores a region.
2. Wild Pheno encounter begins.
3. Player uses Grow Battle actions.
4. Winning or skillful battling awards Cuttings.
5. Cuttings unlock or fill clone recipes.
6. Player starts Clone Dome propagation.
7. Clone timer runs in real time.
8. Rooted Clone reveals trait, expression, stability, and quality.
9. Player adds clone to active team or Vault Garden.
10. Strong clones can be marked as Keepers.
11. Keepers later become breeding/crossing parents.
12. Vault Echo progress and region restoration increase.

## Early Game Loop

Early game teaches:

- exploring
- battling
- earning Cuttings
- rooting clones
- reading simple traits
- updating PhenoLog

The player should not be overwhelmed by genotype or breeding yet.

## Mid Game Loop

Mid game adds:

- Keeper hunting
- expression logs
- genotype hints
- stronger Clone Dome upgrades
- regional restoration rewards
- early Lineage Lab access

## Late Game Loop

Late game adds:

- genotype scanning
- compatible crosses
- seed batches
- hybrid offspring
- environmental breeding
- lineage stabilization
- Legacy/Echo restoration

## Player Reward Types

Players should receive multiple reward categories:

- Cuttings
- Seed Credits
- Trait Samples
- Expression Tags
- Vault Shards
- Clone recipes
- Rooted Clones
- Keeper candidates
- Garden Seals
- Vault Branch restoration
- new areas
- new PhenoLog pages

## No Direct Catching Rule

Wild Phenos are not caught directly. The player earns plant material through battle and roots clones responsibly in the Vault Garden.

This is a critical design rule for originality and cannabis-world logic.

## Story Meaning

Player cloning preserves living genetics through the Vault.

Team Lockout cloning removes variation, corrupts lineages, and creates unstable synthetic clones.

The same core mechanic is used to contrast the hero and villain philosophy.
