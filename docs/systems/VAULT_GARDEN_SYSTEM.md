# Vault Garden System

## System Purpose

The Vault Garden is the player's fictional home base for Rooted Clones, Keeper selection, clone timers, Vault restoration, and future breeding preparation.

It replaces generic creature storage with a living fantasy genetics archive. All actions in this system are game abstractions, not real-world cultivation instructions.

## MVP Functions

The first MVP should include:

- view active team
- view Rooted Clones
- view clone timers
- claim completed clones
- show basic Vault restoration percentage
- mark a clone as a Keeper Candidate
- send clone to active team or Vault Garden

## Full Game Functions

Future versions should include:

- class-specific habitats
- abstract clone-care bonuses
- Stability recovery
- Keeper display areas
- Growth Shift chamber
- Lineage Lab entrance
- genotype scanner
- expression archive
- corrupted quarantine pod
- printable card unlocks
- passive game-item generation

## Habitats

| Habitat | Game Purpose |
|---|---|
| Fruit Grove | boosts Fruit-class trust and expression rolls |
| Root Bed | improves Stability recovery in-game |
| Resin Rock | boosts Frost/Gas expression rolls |
| Quarantine Pod | stores corrupted/synthetic game material safely |
| Legacy Chamber | supports Legacy/Echo progression |
| Clone Dome | completes clone timer cycles from Cuttings |
| Lineage Lab | creates seed batch game outcomes and crosses |

## Clone Dome

The Clone Dome is the main MVP station.

It should allow the player to:

1. select an unlocked clone recipe
2. confirm required Cuttings
3. select optional boost item
4. see current weather/environment influence
5. start a timer
6. check progress
7. reveal the Rooted Clone

## Abstract Care Actions

Care actions are fictional menu choices that modify game stats only.

| Action | Game Bonus |
|---|---|
| Stability Tune | improves Stability roll chance |
| Vigor Boost | improves Vigor roll chance |
| Light Sync | improves expression match chance |
| Timer Boost | slightly reduces game timer |
| Static Cleanse | lowers corruption risk |
| Trait Scan | previews possible trait result |

Care should be optional, not required every timer.

## Save Requirement

Vault Garden save data must track:

- active clone timers
- start timestamp
- duration
- recipe id
- Cutting inputs
- boost item used
- weather/environment at start
- clone slots
- rooted clones
- Keeper flags
- habitat unlocks

## Story Meaning

The Vault Garden is where the player preserves living variation through fictional Vault-connected game systems.

Team Lockout uses sterile synthetic control; the player restores living variation through the Vault.
